There appears to be some confusion with respect to the space for the FAI unique ID numbers on the entry form for the 2018 Indoor World Championship at West Baden. The FAI/CIAM instituted an off cycle rules change that took effect on January 1, 2018. That rule change added the following to section 3.4.2 ofthe F1D rules (actually to the rules for all FAI indoor free flight classes):
 

 "The model shall carry the FAI unique ID number of the competitor on the motorstick written with permanent marker or other non-removable means."
 

 Models will be checked for this during the processing for each round and the ID number on the motor stick should match the ID number submitted by the teams on the entry forms (or in the FAI database).
 

 The 2018 Indoor World Championship is the first FAI sanctioned indoor contest where this rule applies. The 2016 Indoor WC was the first to use the unique ID numbers for pilot identification (but not model identification) and at that event, the FAI provided the ID numbers to the organizers before the contest. We do expect that there may be some problems with both the model markings and the FAI unique ID numbers. As the event approaches, the organizers will have access to the FAI Licence database as will the Jury. We will know the pilots' unique ID numbers as in the database and as on the entry forms so that the models may be processed before each round. We have also made contingency plans in case the model markings are incorrect, not present, or if an entrant has one than one "unique" ID number. The purpose of the rule change is to enforce the BOM rule and prevent more than one pilot from flying the same model. The entrants can rest assured that the officials at the WC will do everything possible to make sure that these checks go smoothly.
 

 The national modeling clubs or the national aeropsport control organizations (NACs) may check the FAI Sporting Licence database at:
 

 https://www.fai.org/sporting-licences https://www.fai.org/sporting-licences

 

 Individuals may obtain an image of their FAI Sporting Licence (which shows their FAI unique ID) at:
 

 http://old.fai.org/about-fai/fai-sporting-licences http://old.fai.org/about-fai/fai-sporting-licences
 

 This information will be in Bulletin 3 but is being posted here so that all may have the information at this time.
