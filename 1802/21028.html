Leo,

On 2/25/2018 4:32 AM, leop_at_lyradev.com [Indoor_Construction] wrote:
>
> The FAI put the marking requirement into the rules that apply to each
> indoor model class and this implies that the models need marking when
> national clubs adopt the FAI rules without change.  However, the FAI
> has informed me that the FAI will interpret the marking rule as only
> applying to FAI sanctioned events.
>

Does this interpretation appear anywhere on the FAI website?  Or in some
official way that the Indoor Contest Board can use as reason to not
enforce the rule unless it is FAI sanctioned event?

> By the way, the JRO is not an FAI sanctioned event.  I doubt the CD,
> Rey Mazzocco, and the chair of the AMA Indoor Contest Board (Don, who
> will be at the JRO) will be enforcing any marking rule for the JRO. 
> And, I doubt that any AMA CD will be enforcing the marking rule in 2018.
>
> .
>
>

That may be true but it needs to be addressed somehow. Simply ignoring a
new rule is not what a CD should be forced to do. Like it or not, the
marking is in the FAI rules now as a model characteristic, so any CD
should enforce the rules as written. Because of how this ID requirement
was written, it is just as important as weighing the motor, checking
wingspan, weighing the model etc. I think if the FAI intended the
marking to apply for only FAI sanctioned events, they need to publicly
post that intent/interpretation on the ID rule ASAP. I think if we have
that then the Indoor Board can use that.
