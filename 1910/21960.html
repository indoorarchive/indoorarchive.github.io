As Yahoo Groups is shutting down, I've been asked to become the owner and moderator for whatever this group is going to become. One existing owner is becoming a moderator and other will stay on as co-owner.

The reason I am writing this is to ask - Where do we go from here?
I joined this group when I got back into indoor FF and loved it for what it is.
Over the years, some of the activity has moved to Hip Pocket's Indoor forum.And more recently, we have seen an uptick in activity on Facebook in Bill/Dima Silin's Indoor Free Flight group.
Given the lack of time we have, it is very likely that we will move to Groups.io
I look at this and think that this same thing must be happening across all FF Yahoo groups.
And I wonder if there is not some way to coordinate this across other FF groups on Yahoo. If you know of groups like this, please let me know. Maybe we can coordinate.

Which gets us to the heart of the matter: None of these offering are really ideal for sharing information. They compete with each other somewhat, each offering having their own strengths and weaknesses. In a sense, we need curated content that is easy to search. We need to be able to ask question, learn from each other and share our knowledge.

So the question becomes: What do we want out of the group?
1. Basic emailed questions/threads? X2. Daily digests?3. Web front end to read/search group threads? (must have good search tool!)
4. File sharing?5. RSS feed?6. What else? Searchable data base/information source X
What do you get out of this group that Hip Pocket and Facebook do not provide?

Anything else I am missing?
Regards.Mike Kirda
