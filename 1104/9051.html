Bill,

For A6 and F1M you will need two timers because they are not listed
contest events. If the sanction was submitted with those events being
flown at the contest then only one timer applies during the contest. So
if you were flying for the LPP record then only one timer needed as that
is listed on the sanction as an event for this contest. I also did
reconfirm with the AMA today that no additional record sanction is
needed, the one existing sanction covers all flying for the day and the
alternate CD has been added to the sanction so everything appears to be
in order.
