LPP is a great model. Nearly everyone starts with an LPP. Some of us don't
ever leave them!

EZB is not for beginners. The heavier F1L is doable but is also not really
an easy model. If you start with LPP and then want to do something more
complicated you can try an F1M which is a very satisfying model with a
weight budget that won't drive you crazy.
