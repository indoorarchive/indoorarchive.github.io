A "Monoject" 28 gauge insulin syringe is pretty handy for thinned 50-50 acetate glues. If you
live in a state that allows these purchased without doctor prescription, these can be bought
in bulk at pharmacies. Maybe other sizes too, such as a bigger gauge that allows filling the
smaller gauge with glue quickly. Momojects last indefinetly if used regularly, are not affected
by acetone, but eventually clog with dry glue, unless you build every day, or want to store it
in jar of acetone. My online source (Countryside Pet Supply) apparently doesn't carry these
anymore (thanks, drug addicts) but they used to be 30c each on sale, in boxes of 100.
