Don,

Only F1D is flown as an FAI championship event. So requiring it only for
the world champs wouldn't make sense either. I would say that since it is
so easy to create an FAI number based on one's AMA number that it would not
be a problem at local contests.

Another way of thinking about it is that most international competitors FAI
numbers are only valid during world championship years. However, it is not
like these numbers disappear or are issued to other competitors during off
years.

Best wishes,

Nick
