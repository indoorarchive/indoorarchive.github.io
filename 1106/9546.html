Kang,

 

I know you're not doing the paperwork, but it is not necessary to apply
separately for an F1D qualifying contest; any sanctioned contest featuring
F1D counts as a qualifying contest per the updated rulebook for 2011. So as
long as it's a contest and 3 F1D fliers are present, the event counts as a
qualifier for the finals. If it's not a contest then yes, we need to apply
for an AMA sanctioned FAI team selection event. But in the past, the 4th has
always been a sanctioned contest.

 

I look forward to seeing everyone for the Freedom1D super-patriotic
spectacular funtabular indoor rubberbonanza luau.
