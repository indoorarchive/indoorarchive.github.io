Thanks for the pointer, I have seen the A6 threads in HPA. I was thinking something more like the Hobby Shop EZB write-up, I don't know if something like that exists for A6.

One specific question, in some indoor threads I see mention of using diluted glue stick dissolved in alcohol as an adhesive for coverings.
This sounds ideal for situations where gnarly solvent based adhesives (3M M77, acetone based diluted glues)  are not usable.  What is a good ratio of glue to alcohol for adhering covering, either tissue or plastics?

I did some experiments with roughly a 50/50 ratio but found that the resulting gloppy stuff was really thick.  It gets thicker as the alcohol evaporates so I've determined this has to be stored in an airtight container but I suspect I can't mix up a big batch and expect it to last for a long time.
Alternatively, are there other adhesives that work well (in an enclosed room) for indoor models?

Any comments would be appreciated.
