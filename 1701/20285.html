I tried looking for a comparison between heptane (Bestine) and naphta to see which is more user friendly.  It looks like both should be used with adequate ventilation, etc... so it doesn't sound like either are going to work well indoors as thinners for rubber cement.
I had been hoping to find an adhesive that I could use indoors with less than adequate ventilation but I guess I'll have to do my covering in the garage with the door cracked open.
