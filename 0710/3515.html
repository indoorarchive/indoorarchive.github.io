I'm not John, but I'll venture a guess.

Usually weights are for the collection of parts unless otherwise
specified. So in your example the weight is for both spars. You might
find something similar for "ribs" which will be for all of the ribs.

Usually for F1D, and F1M, and possibly others, the taper is usually
for just the spars. The tips will be specified separately.

You might want to start with an F1M using a built up prop. The weights
are much easier to build to and an F1M is a bit smaller and definitely
easier to fly.
