This is off-topic, but...

There is an old computer adage that if you don't have two copies of
something, then you really don't have any copies of something. It is
also best to have an off-site backup strategy so that if something
happens to your local machine you have a backup.

For PC's running windows I would suggest something like Carbonite. It
costs $50 a year and backups happen automatically as new files are
created or files are modified.
