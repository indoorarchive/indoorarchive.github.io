I think we need some more fine print so that we can comply with the previous fine print.

1.) The starting time both days is 9:00am (from a previous message). Does this mean that 9:00am is the first time we can launch a flight, having setup earlier? Or is 9:00am the first time we can enter the atrium to setup?

2.) On Saturday, do we need to remove all of our setup and things when we leave by 5:30pm? Or, can we leave things in the atrium overnight?
