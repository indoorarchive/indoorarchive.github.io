Bill,

I think we need some more fine print to help us comply with the current fine print.

1.) The starting time both days is 9:00am (from previous messages). Does this mean we can set up earlier and launch at 9:00am. Or, is 9:00am the time we can first enter the atrium (for example, to setup on Saturday)?

2.) Do we need to remove all of our setup before leaving the atrium on Saturday? If not, what can we leave overnight? If we can leave our tables, chairs, and boxes, how neat do we need to leave things (tools out, etc.)? I know, I am asking the definition of "leave?"

3.) Are spectators allowed? Is there an admission price?

4.) Can we eat and drink in the atrium? If so, can we bring our own food?

More such questions will come up, I am sure.
