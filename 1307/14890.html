On 12.7.2013 22:53, mkirda_at_sbcglobal.net wrote:
> I have some tow used for wrapping carbon capped spars for sailplanes.
>
> Any ideas on how to separate into maybe 25 thread bundles?

My initial response is: just count the 25 threads and pull them apart.

But, I have not seen "raw" wrapping kevlar, so I suppose it has some
binding to keep it together. If so, pulling the separate filaments apart
would be really hard. It would be better to get some untreated rowing,
or a piece of cloth; if the latter pull one thread apart and then start
separating filaments from it. From untreated kevlar it should be easy to
pull the filaments one by one.
