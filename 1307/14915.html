I have posted the Nats/USIC schedule on the AMA website: http://www.modelaircraft.org/files/IndoorNatsschedulev27.pdf

(scroll down to see the Sunday events. I've sent an updated file that fits on one page)

Hopefully this clears up some of the questions I've been receiving:

- FAC events have been expanded, not reduced - and there are more Canone eligible mass launches.

- Glider is still a Nats event

- SO helis can be flown with AMA heli on Fiday, and during the weekend afternoon slots

- SO glider will be flow during the two weekend glider sessions.

BTW - I've heard a rumor that someone is calling for a boycott of the event because their event(s) were removed, and/or the forms were correct initially, and/or the AMA doesn't know how to run an Indoor event.

I sure hope this is not true. If anyone has any concerns, please contact me. There are still plenty of issues that would benefit from a volunteer's time.
