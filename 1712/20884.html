I've been using the same knot for 25 years, and it's never failed. The difference for F1D is I was taught how to tie it in reverse so I can cut my motors to final weight before making the knot. It's an overhand loop knot backed by half a square knot. I use a little hand lotion during the tying process. The lotion provides enough lubrication to prevent chaffing, but not enough to allow the knot to slip. I've tried using silicone oil during knot tying, but the knot frequently slips. The whole process is much easier to demonstrate than it is to describe, so I'm not going to try here.
