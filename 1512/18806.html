At the last St. Lukes contest 2 new records were set. We used the latest form from the AMA website and noticed several problems with the latest version of the form. I suggested some changes to Tony Stillman. Tony jumped on it this week and has revised the form. The new version is not on the website yet but here is a preliminary version. I think the only change from this will be that the phrase (Meet held Outside USA) will be changed to (World Championship held Outside USA).

Changes made:

Page 1
Category heights specified in feet and meters

Page 2
Problem with too many places for official duration cleared up.
For glider records only the two best flights are entered instead of all nine.

Page 3
Area to be filled out by applicant separated from block stating "For Contest Board Use Only"
