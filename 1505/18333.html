This year's AMA Indoor nats has a special event for students where they will build a P-18 model from the http://lasercutplanes.com/ http://lasercutplanes.com/ kit. The students will build the models on Wednesday morning and then fly them Thursday morning. In addition there is an event for all as well.
 

 People can build the kit or scratch build a model.
 The specifications for the scratch built models are:
 1. Projected wing span maximum of 18 inches
 2. Overall model length, including the propeller of 18 inches
 3 propeller is an unmodified 5.5 inch plastic prob. Flashing can be removed and weight added to the light blade for balancing.
 4. Minimum weight of 7 grams.
 

 A pdf file of the full size plans is available from me. drop me an email and I will be happy to send the file.
 

 Have fun with this simple model.
