There are a lot of different opinions about the value of curved wing tips. I've had some people tell me that wing tips should be square. I don't know where the truth is on this. As I see it the chief value of curved tips is to save weight by using smaller spars and fewer glue joints. For an SO model this shouldn't be an issue at all. Personally I think when the area is limited by rules then you should get as much area as you legally can.
