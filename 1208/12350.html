Reminder for everyone: I've seen several questions about making one contest or another be an F1D Team Selection contest qualifying event. Please note that the current program has been modified to allow *any* contest featuring F1D with 3 or more competitors to be a qualifying event. No extra fees or paperwork required for the contest (though you do need to register for the Team Selection program once per cycle).

Just make sure to send me the results so that they can be tabulated. Visit the USAF1D yahoo group to see which qualifying events have been recorded so far. If you don't see yours on there, get me a copy of the results!
