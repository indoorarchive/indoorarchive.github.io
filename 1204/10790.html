The .fhr file is merely a work around since many of our computers warn
us about unknown .exe files. I renamed the .exe file to an .fhr file
(simply my initials) and, after downloading, the person receiving the
file renames it to .exe to run. There is no guarantee that these or any
other files could never cause a problem. I certainly did not knowingly
add any malicious code. These files have been posted several places for
years and may have passed through many hands. I still have a 2001
version on this computer.

Enjoy!
