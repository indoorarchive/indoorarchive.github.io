No. You need to read down the appropriate column (4" for LPP) to 18.9 then go to the left to read the angle which is 37 degrees. Put the prop holder in the end of the gage with 3 holes, put the adjustable part in the set of holes that are 4" from the end ones and set the blade angle at that point to 37 degrees.
