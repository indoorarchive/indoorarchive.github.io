I have taken the AMA record list and edited it for only only the indoor
free flight classes. I have posted the file here in PDF format:
http://www.indoorfreeflight.com/amarecords.pdf
My intention is to have the most recent records on this list and to keep
it up to date so people can just download the file before going to a
contest. To do this I need active support from those who have recently
set records. If a new record is set please email me the information and
I will then update this file and repost it. The list I just uploaded is
the AMA list as of 3/24/2014 , I know a few guys posted some new records
in the past few weeks if you want to send me the details I will update
this file and upload again.
