I've used Duco without trouble. The join is plenty strong for wing post usage. Obviously I wouldn't trust it in a prop hub, but you can afford a couple drops of CA there. I don't recommend ever using CA on motorstick assemblies because you need to be able to easily release whatever glue you've used.
