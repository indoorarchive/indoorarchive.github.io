I'm putting together a list of site records, and I'm hoping the collective
hive mind here can help. I've looked through all of the results I can find
dating back to 1999, and used that to create the list below. I wasn't able
to find results for a few years, and I'm not 100% certain how accurate this
list is right now.

If anyone know of a better time than what is listed below, I would love to
hear it so I can update the records. I'm also not sure if rule changes may
impact any of these events, so let me know if a newer time should be used
for anything.

Also note that I used 2 flight totals for FAI events as well as gliders.

One other note, I used Steve Brown's 1996 F1D flight as the record for
HLS. I'm not sure if that's appropriate, but I want to make sure that time
is recorded because it's the single best flight ever made in the building.

Jake
