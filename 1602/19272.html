Chris has it right. Make an event experience tiered. It worked when I was
shooting competition pistol. After I won 3 times in C class I was bumped
up to B class then finally to A class. Don't make a new class with a
plastic prop that really doesn't teach that much about indoor. Make LPP a
three tier event and allow donated models to be flown in class C or
whatever the beginner class is called. Another option would be to offer
Josh's choice, F1 Max, as a three tiered event.
