I used to get these bottles with the fine metal tubes through my local jeweler as they use them for applying certain adhesives on settings. Great for accelerator. I also discovered if you search the internet using "Needle Oiler Bottles" you can find them easily. The companies that stock these handle gunsmith supplies or model railroading There are plenty of variations so you should be able to pick and choose.
