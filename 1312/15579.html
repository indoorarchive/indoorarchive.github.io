Mike,

I use a 3/4" wood frame with 1/8 aircraft plywood. I use 0.93"Lexan for the
windows as well.

The dimensions are 22" x 9" x 11"

I would recommend avoiding Gator Board it comparatively fragile. I built
one Gator Board box but found it too weak to survive being checked on the
off chance that I had to.

Regards,

Nick
