The Super Sport rubber spreadsheet has been made somewhat more user
friendly and has replaced the previous version in the Files section >
Rubber Motor Calculations.

Anyone wanting to convert their own torque-turns data into a similar
spreadsheet can look in the Yahoo Indoor_Construction Files section at
the Rubber Motor Calculations > Three-Segment Spline folder (May 99),
replace the existing data in the Finder spreadsheet with their own data,
run Solver to fit the curve, and then put the resulting curve parameters
info into a revised User spreadsheet.
