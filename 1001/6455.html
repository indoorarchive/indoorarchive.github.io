Cautery tool tips run up to 2000 deg F. To decrease the temperature without a temp control you will need to reduce the voltage. Most inexpensive cautery tools use one or two AA batteries. Try using only one battery and a spring to replace the second battery. This should reduce the tip temp. Also try using a partially discharged battery. This should reduce the temp even more. It could be a bit of trial and error but somehow you need to reduce the voltage to reduce the tip temp. If you can replace the tip try a larger diameter NiCr wire.

Hope this helps.
