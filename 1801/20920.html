>>> On Jan 7, 2018, at 12:29 AM, dslusarc <dslusarc_at_gmail.com> wrote:
>>>
>>> Dear Indoor Contest Board,
>>>
>>> I want to point out to the Indoor Board two AMA general rule proposals that will have a direct effect AMA indoor records and AMA Indoor events.
>>> Both proposals are listed here.
>>> http://www.modelaircraft.org/events/ruleproposals/indoorff.aspx
>>>
>>> These are listed under Indoor as they are changes to the AMA general rules which apply to all models so Indoor is included.
>>>
>>> IFF 19-01
>>> This will eliminate allowing one flight to set two national records at one time. As a Junior I set records with one flight with a model fitting two classes, such as EZB and Int Stick, 65cm F1D and Hand Launch Stick. This practice would no longer be allowed. Only one class record could be flown for at a time.
>>>
>>> IFF 19-02
>>> This will take AMA events that do not received 10 entries at the Indoor Nats during a 2 year rules cycle and reduce those events from official AMA events to AMA "provisional" status which means national records are no longer kept. After two more years, if still below 10 entries then the event is deleted from the rule book.
>>>
>>> I feel these are important proposals that must be discussed. My initial thoughts are as follows:
>>>
>>> IFF 19-01= Many Indoor records have been set this way over the years by all age classes when planes can fit in dual classes. EZB/Int Stick. Manhattan Cabin/ ROG Cabin, LPP/ OPP, std catapult/unlimited catapult. I see no reason why this needs to change at this point for Indoor. This proposal was written with outdoor FF in mind and is using the general rules to make the changes which will by default effect all Indoor events. The AMA has allowed dual setting of records for a long time. If OutdoorFF wants to change that then that should be taken up by the Outdoor Contest Board and apply to only their events.
>>>
>>> IFF 19-02
>>> This rule proposal is again written for Outdoor FF but is being applied to the general rules so Indoor will be affected. Under these rules most AMA indoor events will be deleted from the rule book in a few rules cycles. Again I feel if Outdoor FF wants to impose such rules on themselves then that needs to be addressed in the Outdoor rules not in the general rules for all models. The Indoor Contest Board is how events are added or deleted, that is our purpose. If the Indoor community feels an event should be deleted it is our responsibility to delete the event as we have done in the past. Not based on event participation at a single contest.
>>>
>>>
