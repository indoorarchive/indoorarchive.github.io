When I looked at the weather report before the contest, it seemed that Saturday might end up being the best air. So busted my butt to get things going as quickly as I could (apologies to anyone I was rude to, I had to fly!). It wasn't easy - we arrived around 9am, had to get set up, and the contest started at noon.

Lucky I did. Today was complete cloud cover, low pressure, strong wind from the east which caused a lot of turbulence and drift. Not good for flying. A few brave souls attempted flights, but Brett and Ben were the only ones to get anything substantial going.

Monday looks like it might be more of the same weather, so it is going to be a real test to see if anyone can work the difficult air. Really big times aren't required - several competitors only need a low 30 to get into the top three.
